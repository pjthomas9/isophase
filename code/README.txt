Fig 8d-f

For each initial condition on spoke and isophase curve, 2,000 CPU were
engaged in parallel. Each generates 300 independent trajectories using
gen_trajs.m (inputs: initial condition (from spoke or isophase curve)
and cluster job id (random number generator seed).

Program calc_frts.m calculates FRT of those trajectories by (i) theta
advances 2pi for spoke initial conditions or (ii) for isophase,
trajectory advances majority rotation but not more than 2 rotations and
nearest distance to isophase points < 1e-2.

See
